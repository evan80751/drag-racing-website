# Drag Racing Personal Website

A personal website about drag racing, featuring information about the sport, my journey, and my garage.

## Features

- Responsive design with Bootstrap 5.3.0
- Interactive JavaScript functionality:
  - Back to top button
  - Dark mode toggle
  - Image hover effects
  - Navigation link animations
- CSS animations for smooth page transitions
- Fully accessible with ARIA labels and semantic HTML
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5.3.0

## Pages

- **Home (index.html)**: Overview of drag racing, history, and famous tracks
- **About (about.html)**: My personal drag racing journey
- **Garage (garage.html)**: Showcase of cars past, present, and future

## Project Requirements Met

- JavaScript page modification (back to top, dark mode, animations)
- CSS animations (fade-in effects, transitions)
- Bootstrap CSS library integration
- Accessibility features maintained
- Hosted on GitHub Pages
